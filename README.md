# 🎮 Phaser Coin Game

A fun and interactive **2D coin collection game** built using **Phaser 3**.  
The player collects coins, avoids challenges, enjoys sound effects and background music, and receives **AI-generated motivational messages** when collecting coins.

---

## 📌 Project Overview

This project is a simple yet engaging browser-based game developed with **Phaser 3**.  
It includes multiple scenes such as a **Menu Scene** and **Game Scene**, and supports a **Continue Game** feature to resume progress.

The main objective of the game is to:
- Move the player character
- Collect coins
- Increase score
- Display AI-generated messages
- Track time and continue saved gameplay

---

## ✨ Features

- 🎮 Built with **Phaser 3**
- 🪙 Coin collection gameplay
- 🤖 AI-generated messages when coins are collected
- 🎵 Background music and sound effects
- ⏱️ Timer system
- 📈 Score tracking
- 🧠 Continue Game feature
- 🧭 Menu Scene and Game Scene
- ⌨️ Keyboard-based player controls
- 🌐 Runs in the browser
